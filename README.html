GETTING STARTED

To run the application you can run the jar from the command line. Make sure you have the version of java with which this solution was built. You can also get the latest version of java here (Latest JDKs).

# Mac OS : set java home to run with java v 25.0.2
$ export JAVA_HOME=`/usr/libexec/java_home -v 25`
$ clear

Running the executable

# Java command
$ java -jar out/Kinomichi.jar
